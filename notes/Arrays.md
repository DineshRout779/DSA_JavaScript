# Time complexities of array methods

| Method  | Time Complexity |
| ------- | --------------- |
| push    | O(1)            |
| pop     | O(1)            |
| shift   | O(n)            |
| unshift | O(n)            |
| concat  | O(n)            |
| slice   | O(n)            |
| splice  | O(n)            |
| sort    | O(n \* logn)    |
| forEach | O(n)            |
| map     | O(n)            |
| reduce  | O(n)            |
| filter  | O(n)            |
